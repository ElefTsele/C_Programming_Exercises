 Practical C programming exercises, organized in a clean, modular structure with optional testing and professional coding practices.

Based on the excellent "{}\+C Programming Examples"{} playlist by \href{https://www.youtube.com/@PortfolioCourses}{\texttt{ Portfolio Courses}}, this repository expands each example with\+:


\begin{DoxyItemize}
\item Modular C code ({\ttfamily .h} / {\ttfamily .c})
\item Robust error handling and status enums
\item Common macros (array size, clamp, swap, etc.)
\item Unit testing setup using Unity
\item Future CI/\+CD integration
\end{DoxyItemize}
